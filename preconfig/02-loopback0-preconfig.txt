Topology를 참조하여 각 네트워크 구간에 IP 주소를 할당하시오
 .GIT Router에 Loopback 0을 생성한후 123.123.123.123/24 IP주소를 할당해야한다.
 .R1 Router에 Loopback 0을 생성한후 11.11.11.11/24 IP주소를 할당해야한다.
 .R2 Router에 Loopback 0을 생성한후 22.22.22.22/24 IP주소를 할당해야한다.
 .ISP Router에 Loopback 0을 생성한후 101.101.10X.X/24 IP주소를 할당해야한다. (X = ISP-X 번호)

	#GIT
en
conf t
interface loopback 0
ip address 123.123.123.123 255.255.255.0
!
end
!
